## 简易应用——Spring Boot会员管理系统

包含功能：

- 会员管理：主要为管理员提供了添加、查询、状态修改（这里有正常、挂失、停用）、会员消费、会员注销功能；
- 商品消费：提供了添加、修改、购买、查询积分记录等功能；
- 礼品消费：礼品设置、数量修改、兑换记录查询等功能；
- 积分抽奖：通过管理员设置抽奖积分然后随机分配给一位会员；
- 生日提醒：通过富文本编辑器提供人性化的邮件提醒服务；
